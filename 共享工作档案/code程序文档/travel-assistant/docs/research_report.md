## 7. 实用贴士

### 最佳旅游时间
- **3-5月**：圆通山樱花盛开，气候宜人
- **9-11月**：秋高气爽，银杏金黄
- **11月-次年3月**：红嘴鸥越冬，观鸥最佳季节

### 注意事项
1. **防晒**：云南紫外线强，需备防晒霜、墨镜、帽子
2. **温差**：早晚温差大，建议带薄外套
3. **高原**：昆明海拔1890米，一般无高反，但避免剧烈运动
4. **饮食**：野生菌需煮熟食用，过桥米线汤烫小心
5. **防坑**：景区门口高价鸥粮不要买，自带面包即可

### 必备APP
- **铁路12306**：高铁购票
- **百度/高德地图**：导航
- **支付宝/微信**：乘车码、支付
- **携程/美团**：酒店、门票预订
