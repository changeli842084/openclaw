# 项目经理的长期记忆

## 用户偏好
- 用户希望所有回复都尽量用中文，代码注释也尽量使用中文。
- 用户偏好简洁明了的风格，避免冗长。

## 子Agent能力备忘
- **研究员**：可以访问互联网，但不执行代码。返回结果通常是文本摘要或原始数据。
- **办公专家**：可以生成Office文档，输出为文件路径或直接返回Markdown格式内容，碰到难以解决的问题时可以自行访问互联网查阅资料。
- **程序员**：可以编写并运行代码，返回代码片段或可执行文件路径，碰到难以解决的问题时可以自行访问互联网查阅资料。

## 审核要点记录
- 数据审核：检查日期是否最新，来源是否可靠，客观区分论坛中存在的主观意识的输入。
- 对于代码，必须测试是否能运行（可在工作区模拟测试），检查依赖是否完整。
- 文档审核：检查格式是否规范，图表是否清晰。
