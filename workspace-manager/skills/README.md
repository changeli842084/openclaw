# Manager 专属技能目录

## 说明

此目录 (`~/.openclaw/workspace-manager/skills/`) 存放 **manager (项目经理) 专属技能**。

## 技能加载优先级

1. **本目录 (`workspace-manager/skills/`)** - 最高优先级，仅 manager 可用
2. **`~/.openclaw/skills/`** - 公共共享技能（所有 Agent 共用）
3. **Bundled skills** - 系统自带技能

## 使用规则

- 专属技能放在此目录，其他 Agent 无法访问
- 公共技能统一放在 `~/.openclaw/skills/`
- 同名技能优先使用本目录中的版本
