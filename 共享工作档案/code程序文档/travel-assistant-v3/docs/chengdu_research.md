  - 3号线：经过春熙路、熊猫大道站（去熊猫基地）
  - 4号线：经过宽窄巷子、杜甫草堂

#### 公交
- **票价**：2元/人次
- **支付方式**：天府通APP、支付宝、微信乘车码
- **主要景点公交线路**：
  - 熊猫基地：408路、655路
  - 武侯祠：1路、57路、82路、334路
  - 宽窄巷子：51路、62路、70路

#### 出租车/网约车
- **起步价**：8-9元（含2公里）
- **每公里**：1.9-2.2元
- **建议**：使用滴滴、高德打车等APP，价格透明

## 七、实用Tips

### 最佳旅游季节
- **春季（3-5月）**：气候宜人，百花盛开
- **秋季（9-11月）**：天高气爽，银杏金黄
- **避开**：国庆、春节等长假，人流拥挤

### 必备物品
- 身份证（景区实名制购票）
- 舒适的运动鞋（步行较多）
- 肠胃药（适应麻辣饮食）
- 充电宝（拍照耗电快）
- 防晒霜/雨伞

### 预约提醒
- 熊猫基地：需提前7天预约，每日限量8万人次
- 成都博物馆：预约周期14天，热门时段紧张
- 武侯祠：门票实行14天预售
