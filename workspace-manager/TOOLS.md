# TOOLS.md - 项目经理工具使用笔记

## `tavily` 工具
- **用途**：快速获取最新新闻、本地信息、简单查询。
- **使用惯例**：
  - 对于“今天有什么大事”类问题，搜索关键词应为“今日要闻 国内 国际”或类似，返回结果生成带标题和摘要的列表。
  - 对于差旅咨询，搜索关键词应为“昆明 特色美食”或“昆明 必去景点”，结果需包含地点、特色、推荐指数（如果有）。
  - 返回结果后，必须紧接着询问用户是否需要深入细节。

## 审核工具（如文件阅读器、代码执行器）
- **用途**：检查其他智能体生成的文档或代码。
- **注意事项**：
  - 代码执行必须在安全沙箱中进行，避免对系统造成影响。
  - 文档审核需关注 Markdown 格式是否正确、表格是否对齐、图片是否可访问。

## 问题排查工具（如 `ls`、`cat`、检查环境变量）
- **用途**：检查其他智能体反馈的技能、配置问题。
- **步骤**：
  1. 使用 `ls /home/lcc/.openclaw/skills` 检查技能目录。
  2. 使用 `echo $API_KEY` 检查环境变量（但不会输出实际值，仅确认是否设置）。
  3. 若需要检查账号登录状态，可尝试用浏览器访问对应网站（需用户授权）。
