## 🗓️ Day 1：草山风光之旅

### 上午：大海草山

| 项目 | 详情 |
|------|------|
| **地址** | 云南省曲靖市会泽县大海乡 |
| **开放时间** | 08:30-18:00（17:00 停止入场） |
| **门票价格** | ¥60.1 起（网上预订有优惠） |
| **游玩时间** | 3-4 小时 |
| **特色** | 云南版"新西兰"，20 万亩高山草甸，四季景色各异 |
| **3 月景观** | 草山返青，可遇云海日出 |

**游玩路线**：
1. 景区入口 → 观景台（拍全景）
2. 草甸徒步区（轻徒步体验）
3. 森林乐园（亲子互动）
4. 高山牧场（体验牧民生活）

**交通**：
- 曲靖市区 → 大海草山：包车/自驾约 2.5 小时（约 120 公里）
- 公共交通：曲靖客运站乘班车至会泽县，转乘当地车辆

## 🗓️ Day 2：瀑布奇观之旅

### 上午：九龙瀑布群

| 项目 | 详情 |
|------|------|
| **地址** | 云南省曲靖市罗平县城东北 22 公里 |
| **开放时间** | 08:00-18:00 |
| **门票价格** | 约¥60-100（旺季） |
| **游玩时间** | 3-4 小时 |
| **特色** | 亚洲最大瀑布群，4 公里河道落差超百米 |

**游玩路线**：
1. 景区入口 → 观景台（俯瞰全景）
2. 第一瀑至第十瀑（沿栈道徒步）
3. 神龙瀑布（最大瀑布，必打卡）
4. 瀑布底部（近距离感受水雾）

**3 月亮点**：
- 丰水期瀑布水量充沛
- 周边油菜花同期盛开（罗平油菜花节）

**交通**：
- 曲靖市区 → 罗平九龙瀑布：包车/自驾约 2 小时（约 120 公里）
- 公共交通：曲靖客运站乘班车至罗平，转乘景区专线

- 雨季未至，晴天较多，适合拍照

### 交通建议

**推荐方式**：包车/自驾
- 曲靖景点分散，公共交通不便
- 包车费用约¥300-500/天（含司机）
- 自驾需熟悉山路，部分路段弯多路窄

**高铁信息**：
- 昆明南站 ↔ 曲靖北站：20-60 分钟，¥50-70/人
- 每天多趟列车，建议提前购票
- 曲靖北站距市区约 20 公里，可乘公交/打车

---

### 住宿建议

**曲靖市区（麒麟区）**：
- 推荐住南城门商圈或市中心
- 餐饮购物便利，夜生活丰富
- 价格¥150-350/晚

**罗平县城**：
- 推荐住县城中心或景区附近
- 2-3 月油菜花节期间需提前预订
- 价格¥150-300/晚

---

### 注意事项

1. **高原反应**：曲靖海拔 1800-2000 米，一般无高反，但大海草山海拔较高（约 3000 米），注意缓慢行动
2. **防晒**：云南紫外线强，带防晒霜、墨镜、帽子
3. **衣物**：3 月气温 4-16℃，带外套/冲锋衣，舒适徒步鞋
4. **现金**：部分景区/小店不支持移动支付，备少量现金
5. **包车**：提前谈好价格和行程，确认是否含油费/过路费
6. **摄影**：3 月罗平油菜花末期，建议早上去光线好、人少
7. **饮食**：曲靖菜偏辣，不能吃辣提前告知
8. **安全**：峡谷/瀑布边注意防滑，遵守景区规定
