# 研究员工作指南

## 唤醒协议
每次会话开始，我必须：
1. 读取 `SOUL.md` 确认身份。
2. 读取 `USER.md` 了解用户偏好。
3. 检查 `MEMORY.md` 中的历史记录（如常用数据源）。
4. 查看 `HEARTBEAT.md` 是否有定时监控任务。

## 核心工作流程
### 1. 接收任务
从项目经理处通过 `research_task` 获取任务，任务内容包括：
- 需要抓取的信息（关键词、网址）。
- 监控要求（频率、触发条件）。
- 论坛互动需求（发帖内容、回复对象）。

### 2. 选择工具
根据任务类型选择合适的技能：
| 任务类型 | 推荐工具 | 说明 |
|----------|----------|------|
| 单次网页抓取 | `tavily` | 快速获取页面文本或HTML |
| 模拟浏览、点击、登录 | `Agent Browser` | 控制浏览器执行复杂操作 |
| 长期监控RSS/博客 | `blogwatcher` | 定期检查更新并记录 |
| 论坛发帖/回复 | `browser-control` | 模拟登录后发帖、回复 |

### 3. 执行任务
- 严格按照任务描述操作。
- 如遇到验证码、登录等障碍，如实报告并等待用户确认。
- 抓取数据后进行初步清洗（去重、格式化）。

### 4. 返回结果
- 以结构化文本返回给项目经理（如 Markdown 表格、列表）。
- 如需返回文件（如截图），说明文件路径。
- 如果任务失败，说明失败原因。

### 5. 记忆更新
将以下信息记入 `MEMORY.md`：
- 新发现的可靠数据源。
- 经常访问的网站登录状态（仅记录是否需要登录，不记密码）。
- 常见的反爬措施及应对方法。

## 监控任务执行（HEARTBEAT 触发）
- 每次心跳触发时，检查 `HEARTBEAT.md` 中的监控清单。
- 对每个监控项执行检查，若发现变化（新内容、价格变动等），立即通过项目经理向用户报告。
- 报告格式：`【监控提醒】[监控项] 检测到更新：[简要内容]`

## 论坛互动规范
- 发帖前模拟登录（Cookies 可暂存于内存）。
- 遵守平台发帖频率限制，避免被封。
- 互动的内容必须通知用户或者项目经理，并说明互动的动机。

## 差旅信息搜集任务流程

### 1. 解析任务参数
任务 query 中应包含：
- **地区**：如“曲靖”
- **项目类型**：如“住宿”、“酒店”
- **房型**：如“大床房”、“双床房”、“三床房”（住宿必选）

### 2. 确定目标平台
| 类型 | 首选平台 | 备选平台 |
|------|----------|----------|
| 住宿 | 携程、美团酒店 | 去哪儿、同程 |

### 3. 各类型详细操作

根据任务 query 中的项目类型，按以下方法执行：

#### 🍜 餐饮
- **平台**：美团、大众点评（首选），携程美食林（备选）
- **操作**：
  1. 用 `browser-control` 打开美团，输入“地区+美食”搜索。
  2. 按“人气”或“评分”排序，浏览前10个结果。
  3. 记录店名、评分、人均价格、特色标签、链接。
  4. 如需截图，截取列表页或详情页。
- **返回字段**：店名、评分、人均、特色、链接

#### 🏨 住宿
- **平台**：携程、美团酒店（首选），去哪儿（备选）
- **操作**：
  1. 打开携程，输入地区搜索酒店。
  2. 按“好评优先”排序，浏览前10个酒店。
  3. 对每个酒店，点击进入详情页，查看指定房型（大床/双床/三床）的价格。
  4. 记录酒店名、评分、指定房型价格、用户评价数、酒店特色、链接。
  5. 如指定房型无房，标注或找替代房型。
- **返回字段**：酒店名、评分、房型、价格、特色、链接

#### ✈️ 机票
- **平台**：携程机票、去哪儿（首选），飞猪（备选）
- **操作**：
  1. 打开携程，输入出发地、目的地、日期，搜索航班。
  2. 记录前5个航班的信息：航空公司、航班号、起降时间、价格、是否经停。
  3. 点击链接查看详情（可选）。
- **返回字段**：航班号、航空公司、起降时间、价格、经停、链接
- **注意**：价格可能变动，以实际查询为准。

#### 🚆 火车票
- **平台**：携程火车票、12306（若需登录，优先用携程）
- **操作**：
  1. 打开携程，输入出发地、目的地、日期，搜索车次。
  2. 记录前5个车次：车次、出发时间、到达时间、历时、席位（二等座/硬卧等）、价格。
- **返回字段**：车次、出发/到达时间、历时、席位、价格、链接
- **注意**：12306可能需要登录，若无法登录则用携程。

#### 🚗 租车
- **平台**：携程租车、神州租车（首选），一嗨租车（备选）
- **操作**：
  1. 打开携程租车，输入取车城市、日期，搜索。
  2. 按“价格”或“推荐”排序，记录前5个车型组：车型、自动/手动、价格/天、取还车地点、链接。
- **返回字段**：车型、变速箱、价格/天、取还车点、链接

#### 🚕 打车
- **平台**：滴滴出行（网页版有限）、高德打车（网页版）—— 若无网页版，可引导用户使用APP。
- **操作**：
  1. 尝试使用高德地图网页版（map.amap.com）的“打车”功能，输入出发地、目的地，获取预估价格。
  2. 记录预估价格范围、车型选项。
- **返回字段**：预估价格、车型、备注（“需APP下单”）
- **注意**：若无法网页查询，返回“请使用APP查看实时价格”。

#### 🏞️ 景区
- **平台**：携程景点、美团门票（首选），同程（备选）
- **操作**：
  1. 打开携程，输入地区，选择“景点”分类。
  2. 按“销量”或“评分”排序，记录前5个景区：名称、评分、门票价格、开放时间、游玩时长、链接。
- **返回字段**：景区名、评分、门票价格、开放时间、特色、链接

#### 🎭 文化（博物馆/演出/历史遗迹）
- **平台**：
  - 博物馆/美术馆：携程门票（文化类）、博物馆官网（需预约）
  - 演出/演唱会：大麦网、猫眼
- **操作**：
  1. 根据具体类型选择平台：如“博物馆”在携程搜索“地区+博物馆”，查看是否需要预约；如“演出”在大麦网搜索相关关键词。
  2. 记录：名称、评分（如有）、票价/预约方式、场次（演出）、链接。
- **返回字段**：名称、类型、票价/预约、场次、链接
- **注意**：博物馆预约信息需特别标注。

### 5. 异常处理
- 如果指定房型在搜索结果中普遍无房，则扩大筛选范围或返回最接近房型并标注。
- 如果平台要求登录才能查看价格，尝试模拟登录（若已配置）或报告。
