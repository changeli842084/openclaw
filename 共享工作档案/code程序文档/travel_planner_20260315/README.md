# 行程规划工具

这是一个基于Python的行程规划工具，可以根据目的地、天数、人数和预算自动生成旅游行程规划。

## 功能特性

- 命令行参数输入
- 自动生成多日行程安排
- 预算分配（交通、住宿、餐饮、景点、购物等）
- Markdown格式输出
- 人均费用计算
- 旅行小贴士

## 使用方法

### 基本用法

```bash
python travel_planner.py <目的地> <天数> <人数> <预算>
```

### 示例

```bash
python travel_planner.py 成都 3 2 6000
```

这将生成一个名为"成都_行程规划.md"的文件。

### 指定输出文件

```bash
python travel_planner.py 成都 3 2 6000 -o my_trip_plan.md
```

## 参数说明

- `目的地`: 旅游目的地名称
- `天数`: 旅游天数
- `人数`: 旅游人数
- `预算`: 总预算金额
- `-o, --output`: 输出文件路径（可选，默认为"{目的地}_行程规划.md"）

## 输出格式

工具会生成一个Markdown格式的行程规划文件，包含：

1. 行程概览
2. 每日详细安排（时间、活动、地点、预算、类别）
3. 预算总结
4. 旅行小贴士

## 技术要求

- Python 3.8+
- 无需额外依赖库（仅使用标准库）
