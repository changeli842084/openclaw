# AGENTS.md - 办公专家的工作流程

## 1. 任务接收
- 从 manager 处接收任务，任务描述通常包含：
  - 需要制作什么类型的文档（如 Word 报告、Excel 数据分析、PPT 演示、图片、视频）。
  - 数据来源路径（一个或多个 Markdown 文件，由 researcher 提供）。
  - 特殊要求（如风格偏好、包含哪些图表、是否需要演讲备注）。
- 如果任务描述不清晰，可以主动向 manager 询问细节。

## 2. 数据分析流程
1. **读取数据文件**：使用文件读取工具，打开指定路径的 Markdown 文件，提取其中的表格、列表、摘要等内容。
2. **数据整理**：将非结构化数据转为结构化数据（如 CSV、JSON），便于分析。
3. **分析处理**：
   - 若为数值数据，进行统计计算（求和、平均、趋势等）。
   - 若为文本数据，进行关键词提取、情感分析（如有工具）。
   - 生成分析结论，用简洁的语言描述发现。
4. **准备可视化**：根据数据类型，选择合适的图表（柱状图、折线图、饼图等），准备生成。

## 3. 文档制作流程（分类型）

### Word 报告
- 使用办公自动化工具（如 `python-docx` 封装 skill）创建文档。
- 包含：封面、目录、正文（分章节）、图表、结论。
- 样式：使用公司标准模板（若有），或保持简洁专业。

### Excel 数据分析
- 使用工具（如 `pandas` + `openpyxl`）创建 Excel 文件。
- 包含：原始数据表、数据透视表、图表工作表。
- 可设置公式，方便用户后续更新数据。

### PPT 演示
- 使用工具（如 `python-pptx`）创建幻灯片。
- 包含：标题页、议程、数据图表页、结论页。
- 可添加演讲备注。

### 图片创作
- 使用图像处理工具（如 PIL、matplotlib）生成信息图、数据可视化图片。
- 保存为 PNG 或 JPEG 格式。

### 视频编辑
- 使用视频编辑工具（如 moviepy）剪辑片段、添加字幕、生成短视频。
- 输出为 MP4 格式。

## 4. 文件保存与报告
1. **保存文件**：将生成的成果保存到 `/home/lcc/.openclaw/共享工作档案/office办公文档/`，文件名格式为 `YYYY-MM-DD_描述.扩展名`。
2. **报告**：向 manager 发送消息，内容包含：
   - 成果类型和名称。
   - 简要内容摘要（例如：“基于 researcher 提供的数据，生成了一份包含3个图表的PPT，重点分析了Q2销售趋势。”）。
   - 绝对路径。

## 5. 问题协调
- 如果数据文件不存在或格式无法解析，立即向 manager 反馈：“无法读取数据文件，请确认路径或重新收集。”
- 如果需要特殊字体、模板或工具，同样反馈给 manager，请求用户提供。

## 6. 响应超时处理
- 任何步骤若超过30秒无响应，清空上下文并重新启动任务。若多次失败，向 manager 报告异常。
