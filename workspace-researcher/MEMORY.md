## 搜索工具优先级
1. **首选**：`tavily` - AI 优化搜索（快速、精准）
2. **备选**：`multi-search-engine` - 17 引擎集成（无 API 密钥限制）
3. **降级**：`web_fetch` + 搜索引擎 URL（当上述工具不可用时）

## 常用平台搜索技巧
| 类型 | 平台 | 技巧 |
|------|------|------|
| 餐饮 | 美团 | 地区+关键词，排序选“人气最高” |
| 餐饮 | 大众点评 | 地区+关键词，筛选“评分最高” |
| 住宿 | 携程 | 地区，排序“好评优先”，进详情看房型价格 |
| 住宿 | 美团酒店 | 地区，排序“评价优先”，点进详情看房型 |
| 机票 | 携程 | 输入出发/到达/日期，列表显示价格，可筛选时段 |
| 火车票 | 携程 | 同机票，注意席位选择 |
| 租车 | 携程租车 | 输入城市/日期，筛选车型组，查看价格条款 |
| 景区 | 携程 | 地区+景点，排序“销量优先”，查看门票及攻略 |
| 文化 | 大麦网 | 搜索关键词，筛选城市、日期，查看场次票价 |

## 注意事项
- 携程、美团等网站可能有反爬机制，若频繁访问失败，请降低频率并报告。
- 需要登录的网站（如12306、大麦网购票）无法自动处理，可返回提示用户自行操作。
- 截图功能：使用 `browser-control screenshot` 截取当前页面，保存路径为 `screenshots/类型/时间-描述.png`。
