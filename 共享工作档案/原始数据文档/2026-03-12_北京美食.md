## 一、著名小吃

### 1. 北京烤鸭 🦆
| 项目 | 详情 |
|------|------|
| **特色** | 皮酥肉嫩、果木烘烤（枣木/梨木），师傅现场片鸭 |
| **吃法** | 搭配饼皮、葱丝、黄瓜条和甜面酱卷食 |
| **历史** | 起源于元代，明清时期为宫廷御膳 |
| **推荐餐厅** | 全聚德（前门店，1864 年）、便宜坊（和平门店，焖炉烤鸭）、大董烤鸭（新派风格） |
| **人均消费** | ¥150-400 |

### 2. 炸酱面 🍜
| 项目 | 详情 |
|------|------|
| **特色** | 黄豆酱 + 五花肉丁慢火熬制，酱香浓郁 |
| **吃法** | 配黄瓜丝、萝卜丝、豆芽、青豆等"小料"拌匀 |
| **推荐餐厅** | 海碗居（人气最高）、柳泉居（老字号） |
| **人均消费** | ¥30-50 |

### 3. 豆汁儿 + 焦圈 🥛

| **推荐餐厅** | 护国寺小吃（百年老字号）、锦芳小吃 |

### 4. 铜锅涮肉 🍲

| **推荐餐厅** | 东来顺（1903 年，王府井）、聚宝源（牛街）、南门涮肉（天坛店）、满福楼 |

## 三、美食集中区域

### 1. 牛街
- **特色**：清真美食聚集地
- **推荐**：聚宝源涮肉、爆肚冯/张、奶酪魏、马辈儿羊汤馆、洪记年糕/包店

### 2. 前门/大栅栏
- **特色**：老字号集中，游客必打卡
- **推荐**：全聚德、天兴居、门框胡同百年卤煮

### 3. 鼓楼/南锣鼓巷
- **特色**：胡同文化 + 小吃
- **推荐**：姚记炒肝、各种胡同小吃摊

### 4. 王府井
- **特色**：商业区，方便游客
- **推荐**：东来顺、四季民福烤鸭

### 5. 天坛附近
- **特色**：本地人常去
- **推荐**：南门涮肉、护国寺小吃

## 四、美食打卡建议路线

### 一日游美食路线
```
早餐：护国寺小吃（豆汁儿 + 焦圈 + 驴打滚）
      ↓
上午：天坛公园游览
      ↓
午餐：南门涮肉（天坛店）
      ↓
下午：故宫/王府井逛街
      ↓
晚餐：四季民福烤鸭（灯市口店）
      ↓
夜宵：姚记炒肝（鼓楼店）
```

### 伴手礼推荐
- **稻香村糕点**：驴打滚、豌豆黄、糖火烧、枣泥饼、豆沙酥
- **吴裕泰茶味冰淇淋**：现场品尝，可买茶味饼干
- **真空包装烤鸭**：全聚德、便宜坊有礼盒装

## 五、注意事项

| 项目 | 提醒 |
|------|------|
| **豆汁儿** | 味道酸涩独特，游客可能不习惯，建议小口尝试 |
| **卤煮火烧** | "重口味"小吃，含猪下水，介意者慎点 |
| **烤鸭** | 建议提前预约，旺季排队时间长 |
| **价格** | 老字号价格透明，景区周边注意辨别 |
| **支付** | 大部分餐厅支持支付宝/微信支付 |
| **营业时间** | 小吃店多早晨开业，部分老店下午早关门 |
